# job-recommendation-app

A web application built using Python, Flask, HTML, and CSS that recommends suitable job roles based on users' technical skills.

## Features

- Skill-based job recommendations
- Interactive web interface
- Dynamic content rendering using Flask
- Personalized career suggestions
- Responsive design

## Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- Git & GitHub

## How It Works

1. Enter technical skills separated by commas.
2. Click "Recommend Jobs".
3. The system analyzes the skills.
4. Relevant job roles are displayed.
